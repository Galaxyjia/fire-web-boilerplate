# fire-web-boilerplate

## 1、安装nodejs和npm
## 2、安装yarn
```
npm install -g yarn
```
## 3、生成vite项目
```
yarn create vite
```
## 4、安装node modules依赖包
```
yarn
```
## 5、运行并查看效果
```
yarn dev
```
## 6、安装tailwind 和 postcss
```
yarn add tailwindcss@latest postcss@latest autoprefixer@latest
```

## 7、生成tailwind和postcss配置文件
```
npx tailwindcss init -p
```
